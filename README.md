## Building from source

Requirements:

- MySQL@^8.0
- Go@^1.23
- Deno@^2.0

> [!TIP]
> First setup:
>
> 1. Run `deno task init:build`.
> 2. Change the `.env` file.
> 3. Run `deno task serve`.
